SeqTree

One of the input files that Corgi uses for training is a SeqTree. This works like a dictionary in Python where the keys are the accessions in the dataset and the value is a SeqDetail which has the index of the cross-validation partition to use for the accession as well as the taxonomic tree that the accession corresponds to.

SeqTree objects can be worked on using seqbank on the command line.

To see the options available, run:

seqbank --help

seqtree

seqtree [OPTIONS] COMMAND [ARGS]...

Options

--install-completion

Install completion for the current shell.

--show-completion

Show completion for the current shell, to copy it or customize the installation.

accessions

Prints a list of accessions in a SeqTree.

If a partition is given, then only the accessions for that partition are given.

seqtree accessions [OPTIONS] SEQTREE

Options

--partition <partition>

The index of the partition to list.

Arguments

SEQTREE

Required argument The path to the SeqTree. <typer.models.TyperPath object at 0x7f8f973c28d0>

export

seqtree export [OPTIONS] SEQTREE SEQBANK OUTPUT

Options

--length <length>

If used then a subsequence this maximum length is selected.

--seed <seed>

The random seed to use when slicing the length.

Default:

42

--partition <partition>

The index of the partition to include in the export. If not given then all accesions will be exported.

--format <format>

The format of the exported file. If not given, then it will be inferred from the file extension of the output.

Default:

Arguments

SEQTREE

Required argument The path to the SeqTree. <typer.models.TyperPath object at 0x7f8f971794d0>

SEQBANK

Required argument The path to the SeqBank. <typer.models.TyperPath object at 0x7f8f97179390>

OUTPUT

Required argument The path to the output file. <typer.models.TyperPath object at 0x7f8f97179b90>

intersection-seqbank

Takes a SeqTree and removes any accession that is not found in a SeqBank.

Saves the output SeqTree to output.

seqtree intersection-seqbank [OPTIONS] SEQTREE SEQBANK OUTPUT

Arguments

SEQTREE

Required argument The path to the SeqTree. <typer.models.TyperPath object at 0x7f8f97326750>

SEQBANK

Required argument The path to the SeqBank. <typer.models.TyperPath object at 0x7f8f9714bd10>

OUTPUT

Required argument The path to the SeqBank. <typer.models.TyperPath object at 0x7f8f972d5010>
